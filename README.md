Task Manager
A simple web application for managing projects and tasks.

Features
 - Create projects
 - Add tasks
 - Track task status
 - User-friendly interface

Technologies
 - React
 - React Router DOM
 - RTK Query
 - TypeScript
 - SCSS
 - JSON Server

Installation
git clone https://github.com/EldarNR/Task-Manager.git
npm install
npm start
npm run server
Then open: localhost
